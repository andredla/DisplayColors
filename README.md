# Display Colors

:. Sublime 3 plugin

Add a preview of the color next to it on code.
Now with real time edit update.
Tested on Linux.


# Instalation guide

- Unzip the file
- Copy the contents to the package folder of Sublime 3.
- Then open Sublime 3 go to View -> Display Colors.
