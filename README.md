# # Sublime 3 - plugin

:. Display Colors

Add a preview of the color next to it on code.
Tested on Linux.


# Instalation guide

- Unzip the file
- Copy the contents to the package folder of Sublime 3.
- Then open Sublime 3 go to Edit -> Display Colors.
